# projetoBootstrap# Projeto Bootstrap - Literatura

Este projeto foi desenvolvido como parte de um trabalho universitário com foco na prática de Bootstrap e desenvolvimento web front end.

O objetivo do site é apresentar, de forma introdutória e acessível, algumas das principais escolas literárias da história da literatura brasileira e portuguesa, abordando seus contextos históricos, características estéticas e principais autores.

A proposta surgiu a partir da importância da literatura para a construção cultural e histórica da sociedade, especialmente no contexto brasileiro. O conteúdo foi organizado de maneira simples e objetiva, permitindo que o usuário tenha um primeiro contato com diferentes movimentos literários e desperte interesse pelo tema.

O desenvolvimento do projeto envolveu a utilização de HTML, CSS e Bootstrap para criação de páginas responsivas, organização de conteúdo e aplicação de componentes modernos de interface.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5.3.8

## Objetivos do Projeto

- Praticar conceitos de desenvolvimento web
- Utilizar componentes e recursos do Bootstrap
- Desenvolver páginas responsivas
- Organizar conteúdo de forma clara e acessível
- Apresentar introduções às principais escolas literárias

## Conteúdos Abordados

O site apresenta páginas sobre movimentos literários como:

- Trovadorismo
- Humanismo
- Classicismo
- Quinhentismo
- Barroco
- Arcadismo
- Romantismo
- Realismo
- Naturalismo
- Parnasianismo
- Simbolismo
- Modernismo
- Pós Modernismo
